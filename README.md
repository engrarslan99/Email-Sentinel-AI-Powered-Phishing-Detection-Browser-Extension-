# Email-Sentinel-AI-Powered-Phishing-Detection-Browser-Extension-
AI-powered browser extension (Chrome, Firefox, Edge) that detects phishing emails in real time using a 3-layer system: NLP classifier (45,000+ emails), Google Safe Browsing API, and rule-based detection. Generates a Trust Score (0-100%) with color-coded risk levels. Works on Gmail and Outlook.
